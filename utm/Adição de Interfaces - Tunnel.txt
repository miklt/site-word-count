['', 'As interfaces túnel são interfaces de firewall que definem uma conexão ponto a ponto para roteadores remotos em redes privadas, e/ou túneis de VPN baseada em rotas.\xa0Qualquer tráfego roteado para uma interface túnel que seja permitido pelas regras de acesso ao Firewall, tende a ser configurado comumente entre um roteador de entrada e de saída, além disso, esse tráfego é enviado para o túnel.O encapsulamento do cabeçalho referente aos protocolos de rede padrão é efetuado nos pacotes cuja função é serem enviados por meio da interface túnel, este encapsulamento através de um cabeçalho novo é denominado GRE (Generic Routing Encapsulation).O protocolo GRE tem por finalidade encapsular uma variedade de protocolos da\xa0camada de rede\xa0dentro de um túnel IP específico, criando uma conexão ponto-a-ponto privada entre roteadores remotos.O protocolo GRE é bastante funcional em vários cenários, normalmente utilizado em VPN para encapsulamento de pacotes enviados através de um túnel pela Internet, funciona de maneira transparente ao usuário, de forma a interligar redes remotas como se estivessem diretamente conectadas.O GRE não é considerado um protocolo seguro, as informações transmitidas através do túnel não são criptografadas, pois o GRE não possui essa função, para corrigir essa lacuna, de modo a garantir a integridade das informações é possível utilizar o protocolo GRE em conjunto com protocolo\xa0IPsec.Funcionamento do protocolo GREO protocolo GRE atua efetuando o encapsulamento de protocolos de camada de rede dentro de conexões virtuais através de um pacote IP externo. Assim sendo, os pacotes encapsulados são transportados através de ambos os pontos do túnel, sendo encaminhados por meio das redes IP. Durante esta rota, os outros roteadores checam apenas o IP externo, não sendo efetuada nenhuma análise do pacote interno até o mesmo ser encaminhado para o destino do túnel GRE. Somente ao atingir seu destinatário no túnel, o pacote tem seu encapsulamento removido e por fim é despachado até seu objetivo.Características do protocolo GRE:Encapsulamento de vários protocolos utilizando apenas um backbone singular;Tráfego multicast e IPv6;Conexão de sub-redes descontínuas;VPN em redes WANs.Para configurar uma interface Tunnel,\xa0 siga os passos a seguir:Clique em\xa0[]\xa0e selecione a opção\xa0[TUNNEL]:Botão de Adição de Interfaces - Menu - TunnelA seguinte tela será exibida:Interfaces - Criação de Interface\xa0TunnelConfigure-a de acordo com as especificações dos respectivos campos, conforme demonstrado a seguir:GeneralInterface Tunnel – General\xa0Network Zone: Este campo refere-se à definição do tipo de agrupamento de interfaces.
