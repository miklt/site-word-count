['', 'O endereço IPv6 possui 128 bits\xa0de comprimento, sendo separado em 8 grupos de 4 caracteres hexadecimais. Além disso, o IPv6\xa0é dividido logicamente em duas partes de 64 bits, a primeira\xa0 para o prefixo da rede e a segunda para identificação da interface.Um endereço IPv6 pode ser de três tipos específicos:Unicast:\xa0Se refere a um único dispositivo, o tráfego de pacotes ocorre pela rota mais curta até o dispositivo;Multicast:\xa0Se refere a um agrupamento de dispositivos, quando o pacote de dados é enviado ao destinatário, cada membro do grupo recebe uma cópia do pacote;Anycast:\xa0Se refere a um agrupamento de dispositivos que possui mesmo prefixo, o pacote de dados é entregue apenas ao remetente mais próximo.Uma das principais diferenças entre o IPv6 e o IPv4 são os múltiplos cabeçalhos de extensão, gerando maior eficiência graças a possibilidade de ajustar seu tamanho, além de ser possível adicionar cabeçalhos novos de forma a atender novas necessidades. Existem seis tipos de cabeçalho de extensão pré-definidos, são eles:Hop-by-hop;Destination Options;Routing;Fragmentation;Authentication;Encrypted security payload.Um host em geral possui dois tipos de IPv6 unicast: o\xa0Endereço de Link Local e o\xa0Endereço Global Unicast. Um endereço global pode ser obtido através dos seguintes métodos de autoconfiguração: SLAAC:\xa0Significa Stateless Address Auto Configuration, através deste método o host utiliza o prefixo do Router Advertising para gerar dois Endereços Globais\xa0Unicast. Neste método o ID da interface é exclusivo na rede local graças ao uso do endereço MAC ou um valor aleatório;DHCPv6 Stateless:\xa0Por meio deste método, o host utiliza o prefixo do Router Advertising para gerar dois Endereços Globais Unicast e utiliza-se de algum servidor DHCPv6\xa0disponível para obter informações relevantes para a configuração, por exemplo o DNS, o Gateway e\xa0o NTP;DHCPv6 Stateful:\xa0Neste método, o host\xa0não consegue obter o prefixo através do Router Advertising, de forma a mitigar isso, é feito um\xa0broadcast\xa0solicitando o Endereço Global Unicast e qualquer outra configuração necessária de um servidor DHCPv6 disponível;DHCPv6 Stateful e SLAAC combinado:\xa0Nesta alternativa, simplesmente aplica o primeiro e o penúltimo método simultaneamente.No sistema, é possível habilitar e configurar o\xa0IPv6, definir como o Router Advertising\xa0será efetuado e executar a conversão de IPv4 para IPv6 através do mapeamento de IP.Para efetuar estas configurações, clique na aba\xa0IPv6 Settings, conforme demonstrado abaixo:Aba\xa0IPv6 SettingsSurgirá a tela a seguir:IPv6 SettingsEsta tela é composta pelos seguintes painéis:IPv6 Settings;Router Advertising;IP address Mapping.A seguir analisaremos cada painel detalhadamente.', '']
